## 📝 Setup & Deployment

This is a GitHub Pages repository. The portfolio is automatically deployed when you push changes to the main branch.

### Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. Edit content and styling as needed

### Enable GitHub Pages
1. Go to repository **Settings**
2. Navigate to **Pages** section
3. Select **Deploy from a branch**
4. Choose **main** branch and **/root** folder
5. Save

Your site will be live at `https://avnish01000.github.io`
